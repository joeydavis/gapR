genGapRDataset.initCenter(center=[50, 50], inSpread=50, r=0, numPoints=5, maxPoints=10, probContinue=0.5, fracContinueDrop=0.5)[source]¶

initCenter is a recursive function that creates new centers for points to be generated at

Parameters:
  • center (list) – The [x,y] location of the center of the points to be distributed about Of the form [x,y]; defaults to [50,50]
  • inSpread (float) – How wide points should be spread about the center Defaults to 50
  • r (int) – a recursion counter to track how deep you’ve recursed Defaults to 0
  • numPoints (int) – The average number of centers/points to generate (real number sampled from a normal dist with u=s=numPoints) Defaults to 5
  • maxPoints (int) – The maximum number of points to be generated Defaults to 10
  • probContinue (float) – The probability that the points will be new centers (another round of recursion) instead of new points Defaults to 0.5
  • fracContinueDrop (float) – The factor by which the probContinue parameter will be dropped if another round of recursion is called Defaults to 0.5
Returns:

a list of lists, each element of the form [x,y,color,recursionLevel]

genGapRDataset.makePoints(numPoints=4, center=[50, 50], spread=50, r=0, ps=0)[source]¶

makePoints is a helper function, it returns a list of points sampled uniformaly parameters described below. NOTE: makePoints updates a global variable COLOR, which is used to keep track of which points were generated together

Parameters:
  • numPoints (int) – The number of points to be generated
  • center – The center of the distribution of points
  • spread (float) – the size of the uniform distribution (each point defined as center+random (0-1)*spread)
  • r (int) – the current recursion depth when the points are made
  • ps (int) – this parameter defines whether COLOR will be updated (the value should only be 1 if you are actually making points, if you are simply using the function to define new centers, ps should be set to 0)
Returns:

a list of lists, each element of the form [x,y,color,recursionLevel]
